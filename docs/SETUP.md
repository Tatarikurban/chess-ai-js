# Установка и запуск

## Установка зависимостей

```bash
npm install
```

## Запуск сервера

Вариант 1 (как раньше):

```bash
node server/server.js
```

Вариант 2 (через npm):

```bash
npm run start
```

После запуска открой `http://localhost:3000`.

## База данных

SQLite создаётся автоматически при первом запуске:

- `server/db/chess.db`

Таблицы:
- `users` — пользователи (никнеймы)
- `games` — лобби/игры
- `moves` — история ходов
