# ONLINE CHESS

проект: онлайн-шахматы (2 игрока) через `socket.io`.

## Возможности
- `index.html` — ввод никнейма
- `lobby.html` — создание/подключение к лобби
- `game.html` — партия (создатель играет **белыми**, подключившийся — **чёрными**)
- SQLite БД: пользователи / игры / история ходов

Смотри `docs/SETUP.md`.
